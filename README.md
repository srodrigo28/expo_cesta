# React Native Expo Exemplos

## Primeiros comandos
* https://cursos.alura.com.br/course/react-native-comecando-zero

* Instalar o nodJS
* Instalar o expo
    * npm install --global expo-cli
* Iniciar o projeto
    * npx expo init react my-project
* Rodando o projeto
    * expo start
* Buildo do projeto
    * expo build: [ios | android]
